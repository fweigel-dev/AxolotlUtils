Axolotl Utils is a client-side Fabric mod designed to enhance the axolotl experience in Minecraft. Whether you're hunting for the elusive blue axolotl or simply want better
visual feedback of the color of it in you bucket, this mod provides quality-of-life improvements that make working with axolotls more enjoyable and efficient.
